Características

Editor de código


El editor de código de Visual Studio admite la configuración de marcadores en el código para una navegación rápida , acepta fragmentos de código, que son plantillas guardadas para código repetitivo las cuales, se pueden insertar en el código y personalizar para el proyecto en el que se está trabajando, también admite la refactorización de códigos, incluida la reordenación de parámetros, el cambio de nombre de variables y métodos, la extracción de interfaces y la encapsulación de miembros de clase dentro de las propiedades, entre otros.
Visual Studio presenta una compilación en segundo plano, a medida que se escribe el código, Visual Studio lo compila en segundo plano para brindar comentarios sobre la sintaxis y los errores de compilación



Depurador


Visual Studio incluye un depurador que funciona como un depurador de nivel de origen y como un depurador de nivel de máquina. Funciona tanto con el código administrado como con el código nativo y se puede usar para depurar aplicaciones escritas en cualquier lenguaje compatible con Visual Studio.
Además, también puede adjuntarse a los procesos en ejecución, monitorear y depurar esos procesos y crear volcados de memoria , así como cargarlos más tarde para la depuración.
Permite establecer puntos de interrupción (que permiten que la ejecución se detenga temporalmente en una determinada posición) y vigila (que monitorea los valores de las variables a medida que avanza la ejecución). Los puntos de interrupción pueden ser condicionales, lo que significa que se activan cuando se cumple la condición. Se puede pasar por alto el código , es decir, ejecutar una línea (del código fuente) a la vez.



Diseñador


Visual Studio incluye una gran cantidad de diseñadores visuales para ayudar en el desarrollo de aplicaciones. Estas herramientas incluyen:


  • Diseñador de Windows Forms

    El diseñador de formularios Windows Forms se utiliza para crear aplicaciones GUI (interfaz gráfica de usuario) utilizando formularios Windows Forms

  • Diseñador de WPF

    • Se utiliza para crear interfaces de usuario dirigidas a Windows Presentation Foundation

  • Diseñador web / desarrollo

    • Se utiliza para desarrollar aplicaciones ASP.NET y es compatible con HTML , CSS y JavaScript

  • Diseñador de clase

    • Se utiliza para crear y editar las clases (incluidos sus miembros y su acceso) mediante el modelado UML

  • Diseñador de datos

    • Se puede utilizar para editar esquemas de bases de datos gráficamente , incluidas tablas tipificadas, claves primarias y externas y restricciones. También se puede utilizar para diseñar consultas desde la vista gráfica.

  • Diseñador de mapas

    • Se utiliza para diseñar el mapeo entre esquemas de base de datos y las clases que encapsulan los datos



Otras Herramientas


  • Abrir el navegador de pestañas

    • Se usa para enumerar todas las pestañas abiertas y para cambiar entre ellas. Se invoca utilizando CTRL+TAB

  • Editor de propiedades

    • Se usa para editar propiedades en un panel de GUI dentro de Visual Studio.

  • Buscador de objetos

    • Se puede usar para explorar los espacios de nombres (que están organizados jerárquicamente) en ensamblados administrados

  • Explorador de la solución

    • Se utiliza para administrar y examinar los archivos en una solución.

  • Explorador de datos

    • Data Explorer se utiliza para administrar bases de datos en instancias de Microsoft SQL Server, también para crear consultas y procedimientos almacenados

  • Explorador de servidores

    • Se usa para administrar las conexiones de la base de datos en una computadora accesible también para explorar los servicios de Windows en ejecución , los contadores de rendimiento, el registro de eventos de Windows y las colas de mensajes y utilizarlos como fuente de datos.

  • Marco de Generación de Texto

    • Permite a Visual Studio generar archivos de texto desde plantillas en el IDE o mediante un código.



Extensibilidad


Visual Studio permite a los desarrolladores escribir extensiones para que Visual Studio amplíe sus capacidades. Estas extensiones se "conectan" a Visual Studio y amplían su funcionalidad.

Las extensiones vienen en forma de macros , complementos y paquetes:

-Los macros representan tareas y acciones repetibles que los desarrolladores pueden grabar mediante programación para guardar, reproducir y distribuir.
-Los complementos brindan acceso al modelo de objetos de Visual Studio y pueden interactuar con las herramientas IDE. Pueden utilizarse para implementar nuevas funciones y pueden agregar nuevas ventanas de herramientas.
-Los paquetes se crean utilizando Visual Studio SDK y proporcionan el nivel más alto de extensibilidad. Pueden crear diseñadores y otras herramientas, así como integrar otros lenguajes de programación
